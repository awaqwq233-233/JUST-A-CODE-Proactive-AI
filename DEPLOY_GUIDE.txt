======================================================================
J.A.C. - Just A Code 智能眼镜助手 - 全套离线/迁移部署指南
======================================================================

本指南用于将 J.A.C. 系统迁移到一台全新的电脑上，支持 Windows 和 macOS 平台。
请按照顺序执行以下步骤。

----------------------------------------------------------------------
步骤 1: 安装 Python 环境
----------------------------------------------------------------------

【Windows 用户】
1. 访问 Python 官网下载 Python 3.10 或 3.11 版本 (推荐 3.10.11 以获得最佳兼容性)：
   下载链接: https://www.python.org/ftp/python/3.10.11/python-3.10.11-amd64.exe

2. 安装时务必勾选：
   [√] Add Python 3.10 to PATH (添加 Python 到环境变量)

3. 安装完成后，打开 PowerShell 或 CMD，输入以下命令检查：
   python --version
   pip --version

【macOS 用户】
1. macOS 通常自带 Python，但建议使用 Homebrew 安装指定版本：
   /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
   brew install python@3.10

2. 安装完成后检查：
   python3 --version
   pip3 --version

3. 建议设置别名（可选）：
   echo 'alias python="python3"' >> ~/.zshrc
   echo 'alias pip="pip3"' >> ~/.zshrc
   source ~/.zshrc

----------------------------------------------------------------------
步骤 2: 手动下载必要的大文件 (模型与工具)
----------------------------------------------------------------------

请提前下载以下文件，并按照说明放入项目文件夹中。

【通用文件】

1. [LLM 大脑模型] (Qwen3.5-9B-Q4_K_M.gguf)
   当前「大脑」模型 Qwen3.5-9B 的 GGUF 已随仓库提供（位于 `J.A.C/models/`），迁移时直接复制整个 `models/` 目录即可；
   运行默认通过 LM Studio (127.0.0.1:12345) 加载，详见步骤 4，无需单独从 HuggingFace 下载。
   * 注意：旧文档里的 Qwen1.5-1.8B-Chat GGUF 已移除，请勿再下载。

2. [Qwen3-TTS 模型] (默认 TTS 语音合成引擎)
   见下方 §5 的一键预下载说明（`python download_models.py`）。模型自动下载到
   `models/qwen_tts/`，离线可用；首次运行也会自动从 HuggingFace 拉取。

3. [YOLOv8 模型] (目标检测 - 默认启用)
   * 首次运行会自动下载，或者手动下载：
   下载链接: https://github.com/ultralytics/assets/releases/download/v8.2.0/yolov8n.pt
   * 将文件放入项目根目录。

4. [LocateAnything-3B 模型] (已移除)
   * 原计划用作视觉语言定位的 NVIDIA 3B 模型，现已从代码中移除。
   * 当前视觉理解仅用 YOLOv8 检测 + LLM 文本摘要，不再下载或加载此模型（约 10GB+，已无用），请勿按旧链接下载。

5. [Qwen3-TTS 模型] (默认 TTS 语音合成引擎)
   J.A.C. 现已默认使用 Qwen3-TTS 进行语音合成（开源、本地优先，支持情绪/语气控制与声音克隆）。

   **一键预下载（推荐）**：运行项目根目录的脚本，自动把模型下载到 models/qwen_tts/ 并离线可用：
     python download_models.py                 # 默认 1.7B-Base（clone 模式）+ 分词器
     python download_models.py --size 0.6B     # 更小的 0.6B（约 2GB）
     python download_models.py --mode custom   # CustomVoice（强情绪控制）
     python download_models.py --mode all      # 该尺寸下三种变体 + 分词器
     python download_models.py --source modelscope   # 国内走 ModelScope（需先 pip install modelscope）
   脚本默认优先 HuggingFace 国内镜像（hf-mirror.com），可用 --no-mirror 关闭。下载走系统 curl（绕开 Python SSL
   证书坑）；若报证书/吊销错误：默认已加 --ssl-no-revoke（跳过 Windows CRL 吊销检查），多数国内网络可直接成功；
   仍失败用 python download_models.py --insecure（关闭校验，仅限可信内网，有中间人风险）。装任何包请用清华镜像：
   pip install -U 包名 -i http://pypi.tuna.tsinghua.edu.cn/simple --trusted-host pypi.tuna.tsinghua.edu.cn

   模型也会在首次运行时自动从 HuggingFace 下载（约 2~4GB）。若已用脚本预下载，
   src/audio/qwen_tts.py 会自动优先加载 models/qwen_tts/ 下的本地副本（离线即可用）。

   手动下载（备用，路径需与脚本一致以便被自动识别）：
   国内推荐 ModelScope（需先 pip install modelscope）：
     modelscope download --model Qwen/Qwen3-TTS-12Hz-1.7B-Base --local_dir ./models/qwen_tts/Qwen3-TTS-12Hz-1.7B-Base
     modelscope download --model Qwen/Qwen3-TTS-Tokenizer-12Hz --local_dir ./models/qwen_tts/Qwen3-TTS-Tokenizer-12Hz
   HuggingFace 方式：
     huggingface-cli download Qwen/Qwen3-TTS-12Hz-1.7B-Base --local-dir ./models/qwen_tts/Qwen3-TTS-12Hz-1.7B-Base

   默认使用 Base 模型 + voices/silverwalf_voice.wav 参考音做 3 秒声音克隆（保住 J.A.C. 音色）；旧参考音 zh_vo_Main_Linaxita_2_1_10_26.wav 保留可回退。
   如需更强情绪控制，可改用 CustomVoice / VoiceDesign，通过环境变量 QWEN_TTS_MODE 与
   QWEN_TTS_MODEL 切换（详见 src/audio/qwen_tts.py 顶部说明）。离线部署时把
   QWEN_TTS_MODEL 指向本地模型目录即可。

【平台特定文件】

【Windows 用户】
4. [Visual Studio C++ 生成工具] (如果安装依赖时报错才需要，通常建议安装)
   下载链接: https://visualstudio.microsoft.com/visual-cpp-build-tools/
   * 安装时勾选 "使用 C++ 的桌面开发"

5. [FFmpeg] (音视频处理核心)
   下载链接: https://www.gyan.dev/ffmpeg/builds/ffmpeg-release-essentials.zip
   * 解压后，找到 bin 文件夹下的 `ffmpeg.exe`。
   * 将 `ffmpeg.exe` 复制到 J.A.C. 项目的根目录下 (与 main.py 同级)。

【macOS 用户】
4. [FFmpeg] (音视频处理核心)
   使用 Homebrew 安装：
   brew install ffmpeg
   
   或者下载预编译版本：
   下载链接: https://evermeet.cx/ffmpeg/
   * 将下载的 `ffmpeg` 文件复制到项目根目录。

----------------------------------------------------------------------
步骤 3: 一键安装依赖
----------------------------------------------------------------------

【Windows 用户】
在新电脑上，打开项目文件夹，在空白处右键选择 "在终端中打开" 或 "Open in Terminal"，
然后复制并粘贴以下所有命令 (一次性粘贴即可)：

# --- 开始 ---

# 1. 升级 pip
python -m pip install --upgrade pip

# 2. 安装基础依赖 (PyTorch, OpenCV, Whisper 等)
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cpu
pip install opencv-python ultralytics pyaudio openai-whisper sounddevice numpy transformers

# 3. 安装 Qwen3-TTS（默认语音引擎）依赖
pip install qwen-tts onnxruntime

# 4. 安装 llama-cpp-python (使用预编译包，避免编译错误)
pip install llama-cpp-python --extra-index-url https://abetlen.github.io/llama-cpp-python/whl/cpu

# 5. 验证 PyAudio (如果上面报错)
pip install pipwin
pipwin install pyaudio

# --- 结束 ---

【macOS 用户】
打开终端，切换到项目目录，然后复制并粘贴以下命令：

# --- 开始 ---

# 1. 升级 pip
python3 -m pip install --upgrade pip

# 2. 安装基础依赖
pip3 install torch torchvision torchaudio
pip3 install opencv-python ultralytics openai-whisper sounddevice numpy transformers

# 3. 安装 PyAudio (macOS 需要 portaudio)
brew install portaudio
pip3 install pyaudio

# 4. 安装 Qwen3-TTS（默认语音引擎）依赖
pip3 install qwen-tts onnxruntime

# 5. 安装 llama-cpp-python
pip3 install llama-cpp-python

# 6. 安装其他依赖
pip3 install pyttsx3 webrtcvad-wheels

# --- 结束 ---

----------------------------------------------------------------------
步骤 4: 运行系统
----------------------------------------------------------------------

【Windows 用户】
在终端中输入：
python main.py

【macOS 用户】
在终端中输入：
python3 main.py

如果一切正常，你应该能看到：
1. 摄像头画面启动
2. 控制台显示 "J.A.C. - Just A Code (多模态版)"
3. 视觉检测器初始化（默认 YOLOv8 物体检测，conf=0.5）
4. 按空格键可以开始对话

【检测模式说明】
- 当前视觉理解仅用 YOLOv8 (conf=0.5) 做物体检测，检测结果 + LLM 文本摘要构成视觉理解；
- 旧文档提到的 LocateAnything-3B / hybrid 模式已从代码中移除，请勿在 main.py 中设置 mode='hybrid'。

----------------------------------------------------------------------
步骤 5: 权限设置 (macOS 特有)
----------------------------------------------------------------------

macOS 要求明确授权应用访问摄像头和麦克风：

1. 首次运行时，系统会弹出权限请求对话框，请点击 "允许"
2. 如果不小心拒绝了，可以在系统设置中修改：
   - 打开 "系统设置" → "隐私与安全性"
   - 在左侧选择 "摄像头"，确保终端或 IDE 被允许
   - 在左侧选择 "麦克风"，确保终端或 IDE 被允许

----------------------------------------------------------------------
常见问题排查
----------------------------------------------------------------------

【下载与证书问题（国内网络常见）】
- Qwen3-TTS 权重下载报 CERTIFICATE_VERIFY_FAILED / CRYPT_E_REVOCATION_OFFLINE：
  脚本已改用系统 curl 且默认加 --ssl-no-revoke（跳过 Windows 证书吊销检查），多数情况可直接成功；
  仍失败用 python download_models.py --insecure 关闭校验（仅限可信内网，有中间人风险）。
- pip 装包（如 qwen-tts）连接 pypi.org 被掐断（SSLEOFError / 超时）：
  改用清华镜像：pip install -U 包名 -i http://pypi.tuna.tsinghua.edu.cn/simple --trusted-host pypi.tuna.tsinghua.edu.cn
- 迁移/克隆后 models/qwen_tts/ 为空：该目录已被 .gitignore 忽略，请运行 download_models.py 重新拉取。

【Windows 常见问题】
- PyAudio 安装失败：使用 pipwin 安装
- ffmpeg 找不到：确保 ffmpeg.exe 在项目根目录
- 模型加载失败：检查模型文件路径是否正确

【macOS 常见问题】
- portaudio 错误：运行 `brew install portaudio`
- 权限错误：检查系统设置中的隐私权限
- Python 版本问题：确保使用 python3 而不是系统自带的 python
- pyttsx3 报错：这是正常的，系统会自动降级到 macOS 内置的 say 命令

【通用问题】
- 模型文件过大：确保有足够的磁盘空间 (约 10GB+)
- 内存不足：建议至少 8GB RAM，推荐 16GB+
- 摄像头无法打开：检查设备连接，尝试更换 camera_id (0, 1, 2...)
